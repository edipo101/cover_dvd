# cover_dvd

Este código escrito en python permite generar covers (cubiertas) DVD a partir de ciertos datos básicos:
- Código de serie
- Url de la imagen principal
- Url de las imagenes de la tapa posterior

## Procedimiento para correr la aplicación
### 1. Crear un entorno virtual y posteriormente activarlo
```python
python -m venv venv
venv\scripts\activate
```
### 2. Instalar las dependencias pip install -r requirements.txt
### 3. Establecer las variables de entorno .env
### 4. Establecer el directorio de salida para los covers ./output
